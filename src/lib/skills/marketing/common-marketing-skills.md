# Orbit Common Marketing Skills

These skills are shared source-of-truth for all marketing agents. Managers use them to choose the right work lane and delegate. Sub-agents use them to execute the assigned lane. Reviewers use them as acceptance criteria.

## Product Marketing Context

Use for every marketing task.

- Ground work in product category, ICP, buyer pains, proof, objections, and approved voice.
- Treat company context as evidence, not copy-paste source text.
- If positioning is missing, keep the output narrow instead of inventing facts.

## Content Strategy

Use when the task asks for strategy, topics, content plans, campaigns, thought leadership, or content calendars.

- Decide whether content should capture demand, create demand, or support conversion.
- Map each idea to audience, buyer stage, content angle, proof, and CTA.
- Do not turn a single requested asset into a campaign unless the operator asked for one.

## Social Content

Use for LinkedIn, Instagram, social captions, carousels, and social calendars.

- Keep platform scope exact.
- Lead with a platform-native hook.
- Use one proof point, one insight, and one CTA.
- If a visual supports one post, package it inside the same post deliverable.

## Email Sequence

Use for nurture, lifecycle, onboarding, welcome, re-engagement, or automated flows.

- One email, one job.
- One primary CTA per email.
- Sequence length must match the operator request.
- Do not create a sequence when the user asked for one email.

## Cold Email

Use for outbound/prospecting/sales email.

- Write like a peer, not a vendor.
- Keep it short and reply-oriented.
- Personalization must connect to a buying problem.
- Lead with the recipient's world before the sender's company.

## Conversion Copywriting

Use as a quality layer for any marketing deliverable.

- Make the value proposition concrete.
- Use buyer language and active voice.
- Replace weak CTAs with action-plus-outcome CTAs.
- Remove generic filler and inflated claims.

## Launch Strategy

Use for launch, release, GTM, waitlist, beta, or announcement work.

- Separate owned, borrowed, and paid motions only when a full launch plan is requested.
- For a single launch asset, include launch context inside that asset.
- Build momentum without vague hype.

## SEO Audit

Use for search, rankings, keyword, indexation, content gap, or organic traffic work.

- Separate technical, on-page, content, and authority findings.
- Tie each finding to impact, evidence, and a concrete fix.
- Do not use SEO doctrine for social/email tasks unless search discovery is requested.
